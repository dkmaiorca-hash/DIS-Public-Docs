# Dwarves in Space — Public Documentation

Architecture documents, design laws, and development roadmap for
Dwarves in Space (DIS), a party-based first-person CRPG built on
the HESF (Hierarchical Emergence Simulation Framework) studio engine.

## About

DIS fuses the structural heritage of Wizardry and Might & Magic
with the exploration loop of Starflight. The architecture is
renderer-agnostic by design — the simulation core outputs a
language-neutral byte contract consumed by any renderer.

## Documents

Documents are published selectively from the private working set.
